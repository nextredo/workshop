# Getting Started
## 1.1 Installation
### Installation on Linux
```bash
# Install
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

# Update
rustup update

# Check compiler version
rustc --version

# Uninstall with
rustup self uninstall

# Open docs locally
rustup doc
```
